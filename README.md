# Todo-ProfileApp
iOS 앱개발 심화

## 🔥필수구현 과제
1. 인스타 프로필 페이지 만들기 -> ProfileDesignViewController (Figma와 동일)
2. 지켜야할 사항
   - UICollectionView ✅
   - UILabel ✅
   - UIImageView ✅
   - UIButton ✅
   - UIView ✅
   - 구성요소 비율, 크기, 여백 구현 ✅
   - 이미지는 Dummy Image 사용 ✅
   - Button Image는 Figma에서 export 후, Asset 형태 사용 ✅
3. Core Data 사용 ✅
